# 🚀 Project:
In this project I learned the concept of SPA (Single-Page Application) and made an application in practice.

- ES Modules;
- Clean Code Concepts;
- Route mapping;
- Asynchronous and promises;
- Object orientation;
- DOM manipulation;
- Web APIs;
- Refactoring; 
- Classes and more.
